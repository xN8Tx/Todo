# TODO App

Simple and beautiful TODO app with json server

## Done with
* React
* Axios
* json-server
* CRA

____

``` npm run start ``` - to start project

``` json-server db.json ``` - to start server

``` npm run build ``` - to build project

*Dont forget to download branch "server"*
